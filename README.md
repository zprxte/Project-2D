## 📖 เกี่ยวกับโครงงาน (About The Project)
โปรเจกต์นี้เป็นการสร้างเกมกระดานปริศนาตัวเลขคลาสสิกอย่าง Sudoku ให้อยู่ในรูปแบบ Web Application โดยทำงานบนฝั่งไคลเอนต์ (Client-side) ทั้งหมด ระบบจะทำการสุ่มสร้างกระดานและซ่อนตัวเลขตามระดับความยากที่ผู้เล่นเลือก ผู้เล่นสามารถโต้ตอบกับกระดานเกมได้ผ่านการคลิกเลือกช่องและเติมตัวเลขที่ถูกต้อง

### ✨ คุณสมบัติหลัก (Key Features)
* 🎚️ **เลือกระดับความยาก (Difficulty Levels):** รองรับ 3 ระดับ ได้แก่ Easy (ง่าย), Medium (ปานกลาง) และ Hard (ยาก) ซึ่งจะกำหนดจำนวนตัวเลขเริ่มต้นบนกระดาน
* 🎲 **ระบบสุ่มกระดานอัตโนมัติ (Dynamic Board Generation):** โค้ด JavaScript จะทำการสุ่มสร้างตารางตัวเลขใหม่ทุกครั้งที่เริ่มเกม ทำให้กระดานไม่ซ้ำกัน
* 🖱️ **ระบบตัวช่วย (Highlight Helper):** เมื่อคลิกที่ตัวเลขบนกระดาน ระบบจะไฮไลต์ (Highlight) ตัวเลขเดียวกันที่อยู่บนช่องอื่นๆ เพื่อช่วยให้เล่นได้ง่ายขึ้น
* 🔢 **แผงควบคุมตัวเลข (Answer Picker):** มีปุ่มตัวเลขด้านล่างสำหรับเติมลงในช่องว่าง โดยปุ่มจะหายไปเมื่อเติมตัวเลขนั้นครบจำนวนในกระดานแล้ว

## 🛠️ โครงสร้างไฟล์ (File Structure)
โครงงานนี้ประกอบด้วย 3 ไฟล์หลัก:

1. `sudoku.html` - โครงสร้างหน้าเว็บหลัก โหลดฟอนต์ (Montserrat) ดึงไฟล์ CSS/JS และเรียกใช้ jQuery
2. `sudoku.css` - ไฟล์ตกแต่งหน้าตาของเกม (UI) รวมถึงการจัดการ Grid ของกระดาน 9x9, อนิเมชัน (Pulse Effect) เมื่อคลิกเลือกช่อง และการออกแบบปุ่ม
3. `sudoku.js` - เป็น Core Logic ของเกมทั้งหมด (เขียนในรูปแบบ jQuery Plugin) ทำหน้าที่สุ่มตาราง, จัดการเหตุการณ์การคลิก, การตรวจสอบคำตอบ และระบบตัวช่วย

## ⚠️ สิ่งที่ต้องเตรียม (Requirements)
* โปรเจกต์นี้มีการใช้ภาพพื้นหลังชื่อ `sudoku.png` คุณจำเป็นต้องมีไฟล์รูปภาพนี้อยู่ในโฟลเดอร์เดียวกับไฟล์ CSS เพื่อให้พื้นหลังแสดงผลได้อย่างถูกต้อง
* จำเป็นต้องมีการเชื่อมต่ออินเทอร์เน็ตเพื่อโหลด jQuery library จาก Google CDN และฟอนต์จาก Google Fonts 

---

## 🚀 วิธีการรันโครงงาน (How to Run)

1. ดาวน์โหลดหรือสร้างไฟล์ทั้ง 3 ไฟล์ (`sudoku.html`, `sudoku.css`, `sudoku.js`) ไว้ในโฟลเดอร์เดียวกัน
2. หาไฟล์รูปภาพที่ต้องการใช้เป็นพื้นหลัง ตั้งชื่อว่า `sudoku.png` และนำมาวางในโฟลเดอร์เดียวกัน
3. ดับเบิลคลิกเปิดไฟล์ `sudoku.html` ผ่านเว็บเบราว์เซอร์ใดก็ได้ (เช่น Chrome, Firefox, Edge)
4. เลือกความยาก (Easy, Medium, Hard) เพื่อเริ่มเล่นเกมได้ทันที
